# An Interactive Map for Delma Island


# Development Practices
## Install both frontend and backend dependencies

```
cd client
npm install
```

```
cd server
npm install
```

## To run front end 
```
cd client
npm start
```
## To run backend 
```
cd server
npm start
```
backend with hot refresh
```
cd server 
nodemon server.js
```
## To run both frontend and backend concurrently
```
cd server
npm run dev

```
